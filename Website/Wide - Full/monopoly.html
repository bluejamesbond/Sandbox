Monopoly: The World Edition

Selected as Best Application in School: Presented in District Technology Fair

My First Project At Age 16

Monopoly world edition features a wide array of options including Java Graphics manipulations, custom painting, and even a fully-optional portable media player that can be changed by coders with only a few steps. This is my first project that I have ever completed. Finished in 11th grade, this was the premonition to my future developments and projects. My computer science teacher had motivated me to push myself to a greater endeavor.

What Is So Great?

This features about 60+ classes that have been maximized both for perfermance and effeciency. There is also an xml driven piece that allows users to edit a few lines to cause large changes. This sort modication allows even the most novice users to adjust the game to their likings. The purpose of this project was purely educational and pushing my personal limits.
