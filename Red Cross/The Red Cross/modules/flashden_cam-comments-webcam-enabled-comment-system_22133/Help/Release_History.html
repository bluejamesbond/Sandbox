The latest release is always available on FlashDen.

Version 1.1

  • New Variable - emailNotification - Optionally receive an email when a user submits a new message. The email contains the users name and message.
  • New Variable - notificationAddress - This coincides with the above variable. If you choose to receive notification emails, use this variable to define where they will be sent.
  • The users webcam is now disabled after submitting a message or closing the write comment window.
  • Fixed an issue where the date on posts made by a user in a timezone ahead of you could read as 'undefined' rather than 'today'.
  • Fixed a hitbox issue with the 'write message' button after submitting a comment.
  • Fixed an issue that could cause a corrupt XML file if a user left a comment before loading the current XML file.
