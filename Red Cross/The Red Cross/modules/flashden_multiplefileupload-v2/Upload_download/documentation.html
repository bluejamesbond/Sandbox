Package contents

The package you downloaded contains the following files:
- documentation.html
- file_upload2.fla
- select_file2.php (where the user gets to select files)
- success.php (shown when the files are uploaded)
- upload.php (background file used to upload)

Modification instructions

Getting things ready
To start uploading all you have to do is upload all files in the zip to a server (except for this documentation file). Be sure your server has php.

Changing the upload folder
You'll need to change two things to choose another upload folder.

1) In the select_file2.php file, on line 3 change $theUploadFolder to the folder you like, for example: $theUploadFolder = "yourfolder";

2) in the upload.php file, on line 3, do the same. These two folder names should be the same!

Variables in the fla file
The following variables are changeable in the fla file.

Name Description
filesizeErrorText The text to be shown when the filesize is too big
uploadErrorText The text to be shown when something is wrong while uploading
spaceTotal Total number of bytes that can be uploaded to the server.
fileTypes Array with filetypes that can be uploaded. description is the value in the selectionbox in the browse window, extension is the allowed extensions.
