Thank you for downloading: TtToolTip 1.0

Included are:

  • F8 Flash File (AS 2.0)
  • TtToolTip.fla

Font(s) used:

  • Futura

How to modify the file:

  1. Change Color:
    Open the MovieClip 'TtColor' (stored in the library) and change its color with the usual Flash tool.

  2. Change Text:
    Change the Value of the Variable '_root.TtText'

  3. Change Rounded Corners:
    Change the Value of the Variable '_root.TtRadius'. Use '0' for no rounded corners

  4. Auto Shrink Corners:
    Do You want the rounded corners to shrink if the ToolTip would be smaller than the rounded corners would be high?
    if yes, set the Variable '_root.shrinkCorners' to true, if not, set it to false

Have Fun using the ToolTip in your projects!
