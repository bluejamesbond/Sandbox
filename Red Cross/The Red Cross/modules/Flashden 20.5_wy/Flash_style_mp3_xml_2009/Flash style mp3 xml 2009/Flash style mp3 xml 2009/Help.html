How to modify the file:

  1. Adding/Removing Music: Open mp3Player.xml and inside the "<music></music>" tags add a new XML item to the list. They will play in this order. The following code should be added. Make sure your title and song file text is inside the <![CDATA[ ... ]]> tags. To delete a song, delete it's XML item from the list.

    <song>
    <title><![CDATA[Title of Song]]></title>
    <mp3File><![CDATA[music/adg3.com_bustedChump.mp3]]></mp3File>
    </song>

  2. Set Player to Automatically Play First Song : Open mp3Player.xml and on line 3 change the "autoPlay" to equal "true". If you do not want it to play automatically set "autoPlay" equal to "false".

  3. Set Player to Automatically Play Next Song :Open mp3Player.xml and on line 3 change the "autoNextSong" to equal "true". If you do not want it to play automatically set "autoNextSong" equal to "false".

  4. Add/Remove buttons: This player is built to easily move around buttons and customize the look-and-feel of the player. If you don't want to have the "next" and "back" buttons showing, simply delete them or set that layer as a "guide" in Flash by right-clicking on that layer and selecting "guide". This applies to all buttons and graphics.

  5. Play a Song Without Using XML Data: If you would like to play a song from another button on your website you simply this code:

    mp3InstanceName.loadMP3("yourFile.mp3","title of song");     //'mp3InstanceName' is the name you apply to the entire mp3 player movie clip on your stage.

  6. Adding to your Flash Site: Copy/paste the movie clip in the main timeline of the given .fla file. Your Flash site must be written in Actionscript 2. Make sure your .xml and music files are in the same directory as this new flash movie you're adding the player to.

  7. Adding to your website: If you are not adding this player to an existing Flash site then you can add this as a stand-alone .swf on your html page. Add the mp3Player_white.swf or mp3Player_black.swf to your html page using Dreamweaver or any html editor that you are comfortable with. Make sure that the mp3Player.xml file and your music files are located in the same location as your mp3Player .swfs.
